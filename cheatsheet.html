SIMD Intrinsics Cheat Sheet

1. SIMD Data Types

Data Type Width (bits) Element Type # of Elements Used For
__m128 128 float (32-bit) 4 128-bit floats (SSE)
__m128i 128 int/char/short Varies (depends) 128-bit integers (SSE/AVX)
__m128d 128 double (64-bit) 2 128-bit doubles (SSE2)
__m256 256 float (32-bit) 8 256-bit floats (AVX)
__m256i 256 int/char/short Varies (depends) 256-bit integers (AVX2)
__m256d 256 double (64-bit) 4 256-bit doubles (AVX)

2. Common Intrinsic Naming Patterns

Naming Part Meaning
_mm Functions working with 128-bit registers (__m128, __m128i, __m128d)
_mm256 Functions working with 256-bit registers (__m256, __m256i, __m256d)
_load Loads data from memory into a SIMD register (aligned by default)
_loadu Loads unaligned data from memory into a SIMD register
_store Stores data from a SIMD register into memory (aligned by default)
_storeu Stores unaligned data from a SIMD register into memory
_add Performs addition on SIMD data
_sub Performs subtraction on SIMD data
_mul Performs multiplication on SIMD data
_div Performs division on SIMD data (typically for floating-point numbers)
_and, _or, _xor Performs bitwise AND, OR, or XOR on SIMD data
_cmpeq Compares SIMD data for equality (returns bitmask of comparison results)
_cmpgt, _cmplt Compares SIMD data for greater-than or less-than
_set Sets a SIMD register with specified values (e.g., _set_epi8 sets bytes)
_set1 Sets a SIMD register with a single repeated value
_max, _min Finds the maximum or minimum value in each lane of SIMD data

3. Data Type Suffixes

Suffix Data Type Example
ps Packed single-precision floating-point (float) _mm_add_ps (adds 4 float values)
pd Packed double-precision floating-point (double) _mm256_add_pd (adds 4 double values)
epi8 Packed 8-bit integers _mm256_add_epi8 (adds 32 8-bit integers)
epi16 Packed 16-bit integers _mm_add_epi16 (adds 8 16-bit integers)
epi32 Packed 32-bit integers _mm_add_epi32 (adds 4 32-bit integers)
epi64 Packed 64-bit integers _mm256_add_epi64 (adds 4 64-bit integers)
si128 128-bit integer register _mm_store_si128 (store 128-bit integer)
si256 256-bit integer register _mm256_load_si256 (load 256-bit integer)

4. Aligned vs. Unaligned Access

Function Operation Description
_mm_load_si128 Aligned load Loads 128-bit from aligned memory into a SIMD register.
_mm_loadu_si128 Unaligned load Loads 128-bit from unaligned memory.
_mm256_load_si256 Aligned load (256-bit) Loads 256-bit from aligned memory into a SIMD register.
_mm256_loadu_si256 Unaligned load (256-bit) Loads 256-bit from unaligned memory.
_mm_store_si128 Aligned store Stores 128-bit from a SIMD register to aligned memory.
_mm_storeu_si128 Unaligned store Stores 128-bit from a SIMD register to unaligned memory.
_mm256_store_si256 Aligned store (256-bit) Stores 256-bit from a SIMD register to aligned memory.
_mm256_storeu_si256 Unaligned store (256-bit) Stores 256-bit from a SIMD register to unaligned memory.

5. Common SIMD Operations Cheat Sheet

Operation 128-bit Function 256-bit Function Description
Add (integers) _mm_add_epi32 _mm256_add_epi32 Adds packed 32-bit integers
Add (floating-point) _mm_add_ps _mm256_add_ps Adds packed single-precision floats
Subtract (integers) _mm_sub_epi16 _mm256_sub_epi16 Subtracts packed 16-bit integers
Multiply (integers) _mm_mullo_epi32 _mm256_mullo_epi32 Multiplies packed 32-bit integers (lower part of product)
Multiply (floats) _mm_mul_ps _mm256_mul_ps Multiplies packed single-precision floats
Bitwise AND _mm_and_si128 _mm256_and_si256 Performs bitwise AND on packed integers
Bitwise OR _mm_or_si128 _mm256_or_si256 Performs bitwise OR on packed integers
Set all to 0 _mm_setzero_si128 _mm256_setzero_si256 Sets all elements to zero
Compare Equal _mm_cmpeq_epi8 _mm256_cmpeq_epi8 Compares 8-bit integers for equality
Compare Greater Than _mm_cmpgt_epi32 _mm256_cmpgt_epi32 Compares 32-bit integers for greater than
Set all elements to 1 value _mm_set1_epi8 _mm256_set1_epi8 Sets all elements in the register to the same byte value
